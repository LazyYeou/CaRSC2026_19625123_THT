#### 2. **Source Control Management**
* a. Commands dalam git: 
    * **git init** : Menginisiasi git repository <br>
    Contoh penggunaan: `git init`
    * **git add** : Memindahkan perubahan pada _working directory_ ke _staging area_ <br>
    Contoh penggunaan: `git add <filename>`
    * **git commit** : Menyimpan perubahan yang ada pada project ke _local git repository history_ <br>
    Contoh penggunaan: `git commit -m "Commit Message"`
    * **git push** : Meng-_upload_ perubahan pada _local repository_ ke _remote repository_ <br>
    Contoh penggunaan: `git push`
    * [**Referensi Git Commands**](https://git-scm.com/docs/git#_git_commands)
